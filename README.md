# RBAC System

This is a **Role-Based Access Control (RBAC) system** built with **Python**, **Flask**, and **SQLite**.  

It demonstrates user roles, permissions, and an API to check access levels in a simulated organization.

## Features

- Users, Roles, and Permissions
- Many-to-many relationship between Roles and Permissions
- API endpoint to check permissions per user
- Sample data: Alice (Admin), Bob (Editor), Eve (Viewer)
